==========================================================
EXPERT VRF-LITE + HSRP (VIRTUAL IP) CONFIGURATION
Architecture: Dual Core/Edge Routers + Layer 2/3 Switch
Feature: High Availability per VRF with Load Sharing
==========================================================

1. ROUTER 1 (R1) - PRIMARY FOR VRF_RED
---------------------------------------
conf t
 vrf definition VRF_RED
  rd 100:1
  address-family ipv4
  exit-address-family
 !
 vrf definition VRF_BLUE
  rd 100:2
  address-family ipv4
  exit-address-family

 interface GigabitEthernet0/1.10
  description *** GATEWAY FOR VRF RED ***
  encapsulation dot1Q 10
  vrf forwarding VRF_RED
  ip address 192.168.10.2 255.255.255.0
  ! HSRP Config for VRF RED
  standby version 2
  standby 10 ip 192.168.10.1
  standby 10 priority 110
  standby 10 preempt
  standby 10 name HSRP_RED
 !
 interface GigabitEthernet0/1.20
  description *** GATEWAY FOR VRF BLUE ***
  encapsulation dot1Q 20
  vrf forwarding VRF_BLUE
  ip address 192.168.20.2 255.255.255.0
  ! HSRP Config for VRF BLUE (Standby here)
  standby version 2
  standby 20 ip 192.168.20.1
  standby 20 priority 100
  standby 20 preempt
  standby 20 name HSRP_BLUE

2. ROUTER 2 (R2) - PRIMARY FOR VRF_BLUE
---------------------------------------
conf t
 vrf definition VRF_RED
  rd 100:1
  address-family ipv4
  exit-address-family
 !
 vrf definition VRF_BLUE
  rd 100:2
  address-family ipv4
  exit-address-family

 interface GigabitEthernet0/1.10
  encapsulation dot1Q 10
  vrf forwarding VRF_RED
  ip address 192.168.10.3 255.255.255.0
  ! HSRP Config (Standby here)
  standby version 2
  standby 10 ip 192.168.10.1
  standby 10 priority 100
  standby 10 preempt
 !
 interface GigabitEthernet0/1.20
  encapsulation dot1Q 20
  vrf forwarding VRF_BLUE
  ip address 192.168.20.3 255.255.255.0
  ! HSRP Config (Active here)
  standby version 2
  standby 20 ip 192.168.20.1
  standby 20 priority 110
  standby 20 preempt

3. ACCESS SWITCH (SW)
----------------------
vlan 10,20
interface range Gi0/1 - 2
 description *** TRUNKS TO R1 & R2 ***
 switchport mode trunk
 switchport trunk allowed vlan 10,20
 spanning-tree portfast trunk

4. ADVANCED VERIFICATION
-------------------------
# Check HSRP Status per VRF:
show standby brief

# Check ARP Table per VRF (to see Virtual MAC):
show ip arp vrf VRF_RED

# Verify Gateway Reachability from a Client:
ping 192.168.10.1 (Should reply from R1)
ping 192.168.20.1 (Should reply from R2)
